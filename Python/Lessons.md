### **Итерация** - однократное выполение тела цикла

### **While(пока)** - пока выполняется одно условие (т.е услоыие истинно) 
```
While условие:
    Команда для выполнения(тело цикла)
    Команда для выполнения(тело цикла)
```

## *Так работает цикл While*
```
age = 0
while age < 18:
    print(f"Мне {age} лет. Я ребёнок!")
    age += 1
```

```
a = 1
b = "фывапол"
while a < 11:
    print(b)
    a += 1
```
### **Это цикл с условием когда мы не знаем точное количество итераций цикла**

### *Конструкция For(для) когда мы знаем сколько будет интераций цикла*

```

```
